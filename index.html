    HTML

    Тэги

    < a > Якорный элемент

    Элемент < a > привязки используется для создания гиперссылок в документе HTML. Гиперссылки могут указывать на другие веб-страницы, файлы на том же сервере, местоположение на той же странице или любой другой URL через атрибут ссылки гиперссылки, href. href Определяет местоположение, на которое указывает элемент привязки.

    < head > Элемент головы

    Элемент < head > содержит общую информацию о странице HTML, которая не отображается на самой странице. Эта информация называется метаданными и включает в себя такие вещи, как заголовок документа HTML и ссылки на таблицы стилей.

    < target > Целевой атрибут

    Атрибут элемента target привязки < a > указывает, где должна открываться гиперссылка. target Значение "_blank"сообщит браузеру, что нужно открыть гиперссылку в новой вкладке в современных браузерах или в новом окне в старых браузерах, или если настройки браузера были изменены для открытия гиперссылок в новом окне.

    < html > HTML-элемент

    Элемент < html > , корень HTML-документа, должен быть добавлен после !DOCTYPE объявления. Весь контент/структура HTML-документа должны содержаться между открывающим и закрывающим < html > тегами.

    Комментарии

    В HTML комментарии могут быть добавлены между открывающим < !--и закрывающим -- >. Содержимое внутри комментариев не будет отображаться браузерами и обычно используется для описания части кода или предоставления других подробностей. Комментарии могут занимать одну или несколько строк.

    < title > Элемент заголовка

    Элемент < title > содержит текст, определяющий заголовок HTML-документа. Заголовок отображается в строке заголовка браузера или на вкладке, на которой отображается HTML-страница. Элемент < title > может содержаться только внутри элемента документа < head >< /head >

    < li > Элемент списка элементов

    Элемент списка < li > создает элементы списка внутри:

    • Упорядоченные списки < ol >
    • Неупорядоченные списки < ul >

    < video > Видео элемент

    Элемент < video > встраивает медиаплеер для воспроизведения видео. src Атрибут будет содержать URL-адрес видео. Добавление controls атрибута отобразит элементы управления видео в медиаплеере.
    Примечание: содержимое внутри открывающего и закрывающего тега отображается как резервный вариант в браузерах, которые не поддерживают этот элемент.

    < em > Элемент акцента

    Элемент < em > выделяет текст, и браузеры обычно по умолчанию выделяют выделенный текст курсивом.

    < div > Элемент Div

    Элемент < div > используется как контейнер, который разделяет HTML-документ на разделы и является сокращением от «division» (разделение). < div > Элементы могут содержать потоковое содержимое , такое как заголовки, абзацы, ссылки, изображения и т. д.

    < br > Элемент разрыва строки

    Элемент < br > разрыва строки создаст разрыв строки в тексте и особенно полезен там, где требуется разделение текста, например, в почтовом адресе. Элемент разрыва строки требует только открывающего тега и не должен иметь закрывающего тега.

    < img > Элемент изображения

    Элементы HTML-изображений < img > встраивают изображения в документы. src Атрибут содержит URL-адрес изображения и является обязательным. < img >— пустой элемент, то есть он не должен иметь закрывающего тега.

    < p > Элемент абзаца

    Элемент < p > абзаца содержит и отображает блок текста.

    < body > Элемент тела

    Элемент < body > представляет содержимое HTML-документа. Содержимое внутри < body > тегов отображается в веб-браузерах.
    Примечание:< body > в документе может быть только один элемент.

    < span > Элемент пролета

    Элемент < span > представляет собой встроенный контейнер для текста и может использоваться для группировки текста в целях стилизации. Однако, как и < span > универсальный контейнер для отделения фрагментов текста от более крупного текста, его использования следует избегать, если доступен более семантический элемент.

    < strong > Сильный элемент

    Элемент < strong > выделяет важный, серьезный или срочный текст, и браузеры обычно по умолчанию отображают этот выделенный текст жирным шрифтом

    Таблицы

    < table > Элемент таблицы

    В HTML < table > элемент имеет содержимое, которое используется для представления двумерной таблицы, состоящей из строк и столбцов.

    < thead > Элемент заголовка таблицы

    Элемент заголовка таблицы < thead > определяет заголовки столбцов таблицы, заключенные в строках таблицы.

    < tbody > Элемент тела таблицы

    Элемент тела таблицы, < tbody >, является семантическим элементом, который будет содержать все данные таблицы, кроме заголовка таблицы и содержимого нижнего колонтитула таблицы. Если используется, < tbody > будет содержать все элементы строк таблицы < tr > и указывает, что < tr > элементы составляют тело таблицы. < table > не может иметь < tbody >и < tr > как прямых потомков.

    < tfoot > Элемент нижнего колонтитула таблицы

    Элемент нижнего колонтитула таблицы < tfoot > использует строки таблицы для указания содержимого нижнего колонтитула или для суммирования содержимого в конце таблицы.

    < tr > Элемент строки таблицы

    Элемент строки таблицы, < tr >, используется для добавления строк в таблицу перед добавлением данных таблицы и заголовков таблицы.

    < td > Элемент данных таблицы

    Элемент данных таблицы, < td >, может быть вложен в элемент строки таблицы, < tr >, чтобы добавить ячейку данных в таблицу.

    < th > Элемент заголовка таблицы

    Элемент заголовка таблицы, < th >, используется для добавления заголовков к строкам и столбцам таблицы и должен быть заключен в элемент строки таблицы, < tr >.

    rowspan Атрибут

    Подобно colspan, rowspan атрибут заголовка таблицы или элемента данных таблицы указывает, сколько строк должна охватывать эта конкретная ячейка в таблице. Значение rowspanпо умолчанию равно 1 и может принимать любое положительное целое число до 65534.

    Формы

    < input >: Тип флажка

    input При использовании элемента HTML type="checkbox"атрибут будет отображать один элемент флажка. Чтобы создать группу флажков, относящихся к одной теме, они все должны использовать один и тот же name атрибут. Поскольку это флажок, для одной темы можно выбрать несколько флажков.

    < textarea > Элемент

    Элемент textarea используется при создании текстового поля для многострочного ввода (например, раздел комментариев). Элемент поддерживает атрибуты rowsи cols, которые определяют высоту и ширину элемента соответственно. При отображении в браузере textarea поля могут быть растянуты/уменьшены пользователем, но атрибуты rowsи cols определяют начальный размер. В отличие от input элемента, < textarea > элемент имеет как открывающий, так и закрывающий теги. Элемент value элемента — это содержимое между этими тегами (очень похоже на < p > элемент).

    < form > Элемент

    Элемент HTML < form > используется для сбора и отправки информации во внешний источник. < form > может содержать различные элементы ввода. Когда пользователь отправляет форму, информация в этих элементах ввода передается источнику, который указан в action атрибуте формы.

    < input >: Тип номера

    Элементы ввода HTML могут иметь тип number. Эти поля ввода позволяют пользователю вводить внутри поля только цифры и несколько специальных символов. В примере блока кода показан ввод с типом numberи именем balance. Когда поле ввода является частью формы, форма получит пару ключ-значение в формате: name: value после отправки формы.

    < input > Элемент

    Элемент HTML < input > используется для отображения различных полей ввода на веб-странице, включая текстовые поля, флажки, кнопки и т. д. < input > У элемента есть type атрибут, который определяет, как он отображается на странице.Пример блока кода создаст поле ввода текста и поле ввода флажка на веб-странице.

    < input >: Тип диапазона

    Ползунок можно создать, используя type="range"атрибут в элементе HTML input. Ползунок диапазона будет действовать как селектор между минимальным и максимальным значением. Эти значения задаются с помощью атрибутов minи max соответственно. Ползунок можно настроить для перемещения с различными шагами или приращениями с помощью step атрибута. Ползунок диапазона предназначен скорее для работы в качестве визуального виджета для регулировки между 2 значениями, где относительное положение важно, но точное значение не так важно. Примером этого может быть регулировка уровня громкости приложения.

    < select > Элемент

    Элемент HTML < select > может быть использован для создания раскрывающегося списка. Список вариантов для раскрывающегося списка может быть создан с использованием одного или нескольких < option > элементов. По умолчанию < option > за раз можно выбрать только один.Значение выбранного атрибута < select >'s name и < option >'s value отправляются как пара ключ-значение при отправке формы.

    Отправка формы

    После того, как мы собрали информацию в форме, мы можем отправить ее куда-то еще, используя атрибут actionand method. action Атрибут сообщает форме, что нужно отправить информацию. Назначается URL, который определяет получателя информации. Атрибут method сообщает форме, что делать с этой информацией после ее отправки. HTTP-глагол назначается атрибуту method, который определяет действие, которое необходимо выполнить.

    < input > : Тип текста

    HTML- < input > элементы могут поддерживать ввод текста, установив атрибут type="text". Это отображает однострочное поле ввода, в котором пользователи могут вводить текст.Значение атрибута < input > и name элемента value отправляются как пара ключ-значение при отправке формы.

    < datalist > Элемент

    При использовании ввода HTML базовая функциональность поиска/автозаполнения может быть достигнута путем сопряжения с < input >. < datalist >Для сопряжения < input > с ' значение ' < datalist > должно совпадать со значением ' . Элемент используется для хранения списка '.< input >listid < datalist > datalist< option > Список данных отображается в виде раскрывающегося списка в input поле, когда пользователь нажимает на поле ввода. Когда пользователь начинает печатать, список будет обновляться, чтобы показать элементы, которые лучше всего соответствуют введенному в поле ввода. Фактические элементы списка указываются как несколько option элементов, вложенных в datalist. datalist идеально подходят для предоставления пользователям списка предопределенных вариантов, но также позволяют им вводить альтернативные входные данные.

    < input >: Тип радиокнопки

    Элементам HTML < input > можно задать type="radio"атрибут, который отображает одну кнопку-переключатель. Нескольким кнопкам-переключателям связанной темы задается одно и то же name значение атрибута. Из группы кнопок-переключателей можно выбрать только один вариант.Значение выбранных/отмеченных < input > элементов nameи value атрибут этого элемента отправляются как пара ключ-значение при отправке формы.

    Представляемый ввод

    Элементы HTML < input > могут иметь атрибут type, установленный для отправки, путем добавления type="submit". При включении этого атрибута кнопка отправки будет отображаться и, по умолчанию, будет отправлять < form > и выполнять свое действие. Текст кнопки «Отправить» установлен Submit по умолчанию, но его также можно изменить, изменив value атрибут.

    < input > name Атрибут

Это достигается путем установки атрибута name элемента input. name Станет , key а value ввода станет форма value отправляет соответствующие ключу. Важно помнить, что имя не то же самое, что и ID с точки зрения отправки формы. ID и имя ввода могут быть одинаковыми, но значение будет отправлено только в том случае, если name указан атрибут.     < label > Элемент

    Элемент HTML < label > обеспечивает идентификацию для определенного < input > на основе соответствия значений атрибута < input >' idи атрибута < label > ' for. По умолчанию нажатие на < label > будет фокусироваться на поле связанного < input >. Пример кода создаст поле ввода текста с текстом метки «Пароль: «рядом с ним. Нажатие на «Пароль: «на странице переместит фокус на поле для связанного < input >.

    < input > Тип пароля

    Элемент HTML < input > может иметь атрибут type="password", который отображает поле ввода с одной строкой, что позволяет пользователю вводить цензурированный текст внутри поля. Он используется для ввода конфиденциальной информации. Значение этого < input > атрибута nameи value(фактическое значение, а не цензурированная версия) этого элемента отправляются как пара ключ-значение при отправке формы. В блоке кода показан пример полей для базовой формы входа в систему — поля имени пользователя и пароля.

    required Атрибут

    В HTML поля ввода имеют атрибут required, который указывает, что поле должно содержать значение. примере блока кода показано поле ввода, которое является обязательным. Атрибут может быть записан как required="true" или просто required.

    max Атрибут

    HTML < input > -элементы типа number имеют атрибут max, который указывает максимальное значение для поля ввода. Блок кода показывает input числовое поле, для которого установлено максимальное значение 20. Любое значение больше, чем 20 будет отмечать поле ввода как имеющее ошибку.

    maxlength Атрибут

    В HTML поля ввода с типом text имеют атрибут maxlength, который определяет максимальное количество символов, которые можно ввести в поле. Блок кода показывает поле ввода текста, которое принимает текст длиной не более 140 символов.

    pattern Атрибут

    В text элементе ввода pattern атрибут использует регулярное выражение для сопоставления со значением (или проверки) < input > при отправке формы.

    minlength Атрибут

    В HTML поле ввода типа text имеет атрибут, который поддерживает проверку минимальной длины. Чтобы проверить, что текст ввода имеет минимальную длину, добавьте minlength атрибут с количеством символов.

    Валидаторы HTML-форм

    HTML-формы позволяют вам указывать различные виды проверки для ваших полей ввода, чтобы убедиться, что данные введены правильно перед отправкой. HTML поддерживает ряд различных валидаторов, включая такие вещи, как минимальное значение, минимальная/максимальная длина и т. д. Валидаторы указываются как атрибуты поля input.

    min Атрибут

    В HTML поля ввода с типом numberимеют атрибут min, который определяет минимальное значение, которое можно ввести в поле.

    CSS

    < link > Элемент ссылки

    Элемент используется для связывания HTML-документов с внешними ресурсами, такими как файлы CSS. Обычно он использует:

    • href атрибут для указания URL-адреса внешнего ресурса
    • rel атрибут для указания связи связанного документа с текущим документом
    • type атрибут для определения типа связанного контента

    Селекторы классов и идентификаторов

    Классы CSS можно использовать повторно и применять ко многим элементам. Селекторы классов обозначаются точкой, .за которой следует имя класса. Селекторы идентификаторов CSS должны быть уникальными и использоваться для стилизации только одного элемента. Селекторы идентификаторов обозначаются знаком решетки, #за которым следует имя идентификатора.

    Цепочка селекторов

    Селекторы CSS можно объединять в цепочку, чтобы наборы правил применялись только к элементам, которые соответствуют всем критериям. Например, чтобы выбрать < h3 > элементы, которые также имеют класс, можно использовать section-heading селектор .h3.section-heading

    Селекторы и классы

    CSS-декларации

    В CSS декларация — это пара ключ-значение свойства CSS и его значения. Декларации CSS используются для установки свойств стиля и построения правил, применяемых к отдельным элементам или группам элементов. Имя свойства и значение разделяются двоеточием, а вся декларация должна заканчиваться точкой с запятой.

    Размер шрифта

    Свойство font-size CSS используется для установки размеров текста. Значения размера шрифта могут быть разными единицами или типами, такими как пиксели.

    Цвет фона

    Свойство background-colorCSS управляет цветом фона элементов.

    !important Правило

    Правило CSS !important используется в объявлениях для переопределения любых других объявлений для свойства и игнорирования специфичности селектора. !importantправила гарантируют, что определенное объявление всегда применяется к соответствующим элементам. Однако, как правило, лучше избегать использования !importantв качестве плохой практики.

    Непрозрачность

    Свойство opacityCSS может использоваться для управления прозрачностью элемента. Значение этого свойства варьируется от 0(прозрачный) до 1(непрозрачный).

    Насыщенность шрифта

    Свойство font-weightCSS может использоваться для установки веса (жирности) текста. Указанное значение может быть ключевым словом, например boldили normal.

    Выравнивание текста

    Свойство text-alignCSS можно использовать для установки выравнивания текста встроенного содержимого. Этому свойству можно задать следующие значения: left, right, или center.

    Наборы правил CSS

    Набор правил CSS содержит один или несколько селекторов и одно или несколько объявлений. Селектор(ы), который в этом примере является h1, указывает на элемент HTML. Объявления, которые в этом примере являются color: blueи text-align: centerстилизуют элемент с помощью свойства и значения. Набор правил является основным строительным блоком таблицы CSS.

    Установка цвета текста переднего плана в CSS

    Используя color свойство, цвет текста переднего плана элемента может быть установлен в CSS. Значением может быть допустимое имя цвета, поддерживаемое в CSS, например greenили blue. Также для установки цвета можно использовать 3-значный или 6-значный код цвета, например #22fили .#2a2aff

    URL-адреса ресурсов

    В CSS эта url()функция используется для оборачивания URL-адресов ресурсов. Они могут применяться к нескольким свойствам, таким как background-image.

    Фоновое изображение

    Свойство background-imageCSS устанавливает фоновое изображение элемента. URL-адрес изображения должен быть указан в синтаксисе url("moon.jpg")как значение свойства.

    Семейство шрифтов

    Свойство font-familyCSS используется для указания гарнитуры шрифта в наборе правил. Шрифты должны быть доступны браузеру для корректного отображения, либо на компьютере, либо связаны как веб-шрифт. Если значение шрифта недоступно, браузеры будут отображать свой шрифт по умолчанию. При использовании имени шрифта из нескольких слов лучше всего заключать их в кавычки.

    Название цвета Ключевые слова

    Ключевые слова названий цветов можно использовать для установки значений свойств цвета для элементов в CSS.
